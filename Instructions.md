# Key-Breaker Instructions

## Windows:
CMD:
```bash
#install 
git clone https://github.com/KazutoKashima/Project-Key-Breaker.git
cd Project-Key-Breaker/bin/
powershell.exe
```
PowerShell:
```powershell
cd path/to/project-key-breaker
./windows.ps1

# Run keybreaker:
npm start win chrome
```

## Linux:
```bash
git clone https://github.com/KazutoKashima/Project-Key-Breaker.git
cd Project-Key-Breaker/bin/
sudo bash linux-apt.sh

# Runnng keybreaker
npm start win chrome
```
